> [Задача 39](https://github.com/XYI7I/GeekBrains/blob/main/Geek/C%23/lesson6/task1/Program.cs): Напишите программу, которая перевернёт одномерный массив (последний элемент будет на первом месте, а первый - на последнем и т.д.)
